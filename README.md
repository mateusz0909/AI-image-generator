
# AI Image Generator

Simple app that allows users to generate image purely by AI. Image maker is powered by [OpenAI](https://openai.com/dall-e-2/)

## Tech Stack

**Client:** React, CSS

**Server:** Express.js



## Features

- User can create realistic images and art from a description in natural language
